minimal neovim config for use with termux, or without

## setup
- backup your own config
- clone this repository 
  ```
  git clone https://github.com/pygmysrc/mini-config.git
  ~/.config/nvim
  ```
- start nvim

## key mappings

| key                         | code                |
| :---                        | ---:                |
| `s`                         | `save`              |
| `q`                         | `quit`              |
| `H`                         | `bprev`             |
| `L`                         | `bnext`             |
| `z`                         | `only`              |
| `_e`                        | `files`             |
| `_d`                        | `bdelete`           |
| `_w`                        | `save`              |
| `_q`                        | `save / quit`       |
